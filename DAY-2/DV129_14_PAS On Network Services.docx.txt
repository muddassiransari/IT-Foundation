DV129_14_PAS On Network Services
Self-Assessment Sheet


1. What is a Web Server?
Ans: A Web server is a computer or software application that stores and serves website content to user over the internet.        
2. What is a File Server?
Ans: A File server is a specialized computer or software application designed to store,manage and provide access to files and data for clients within a network.        
3. What is Print Server? 
Ans: A print server is a device or software application that manages and controls printing tasks on a network. its primary purpose is to centralize and streamline the process of sending print jobs from multiple clients to one or more printers. 
4. What is a (Domain Name System) DNS Server?
Ans: The Domain Name System server is crucial component of the internet infrastructure that translate human-readable domain names like www.professormesser.com into numerical IP addresses that computers use to identify and communicate with each other. 
5. What is a (Dynamic Host Configuration Protocol) DHCP Server?
Ans: The Dynamic Host Configuration Protocol server is a network server that automatically assigns IP addresses and network configuration settings to devices on local network. It’s a crucial component in simplifying and automating the process of configuring IP addresses and related network parameters for devices, such as computers, smartphone and other networked devices.
6. What is a Mail Server?
Ans: A Mail server is a specialized computer or software application that handles the sending, receiving and storage of email messages within a network. 
7. What is a Proxy Server?
Ans: A proxy server is an intermediary server that acts as gateway between a client and other servers, including Web servers, in a network. It servees as point of contact for request and responses, helping to enhance security, performance and privacy for users and the network as a whole. 
8. What is an Authentication Server?
Ans: An authentication server is a specialized server that validates the identify of users or devices attempting to access a network or a specific system. It plays a critical role in ensuring that only authorized users or devices can gain access to resource, services or applications within a networked environment.         
9. What is SIEM (Security Information and Event Management)?
Ans: Security Information and Ecent Management is Comprehensive approach to managing an organization’s information security by centralizing the collection, analysis and correlation of security- related data from various sources. 
10. What is Forensic Analysis?
Ans: Forensic analysis also known as digital forensics is systematic process of investigating and analyzing digital evidence to uncover information related to cybercrimes, security incidents legal ceses or any other situation where digital data needs to be examined to understand what happened who was involved and how it occurred. 
11. What is Syslog?
Ans: Syslog is a standard protocol used for sending used for sending and receiving log message and event notification within a computer network. It is commonly used for centalizing and managing log data generated by various devices and application in a networked environment.  
12. What is IDS (Intrusion Detection System)?
Ans: An intrusion Detecion System is a security technology designed to monitor network traffic or system activities fir signs of unauthorized or malicious activities. 
13. What is IPS(Intrusion Prevention System)?
Ans: An Intrusion Prevention System is a security technology designed to not only detect but also actively prevent and block unauthorized or malicious activities on a network or system 
14. What is an All-in-one security appliance?
Ans: An all-in-one security appliance often referred to as a Unified Threat Management appliance is a hardware or virtual device that integrates multiple security functions into a single solution. 
15. What is Endpoint Management Server?
Ans: Endpoint Managment Server is a centralized software solution designed to manage and control various endpoint within an organization’s network. 


16. What is a Legacy System?
Ans: A Legacy System refers to outdated or older computer system, software application, hardware components or technologies that have been in use for a significant period of time and are no longer actively developed or maintained by their original creators. 
17. What is an Embedded System?
Ans: An embedded system is a specialized computing system designed to perform specific function or tasks within a larger system often as part of a larger device or product . unlike general purpose computers which can run a wide range of software application embedded system are tailored to perform dedicated tasks and are optimized for efficiency, reliability and real time operation.
